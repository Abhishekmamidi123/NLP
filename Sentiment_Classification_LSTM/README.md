### Sentiment Classification on IMDB Dataset:
- Framework used: [Keras](https://keras.io/)
- Implemented using two different architechtures:
  - Using LSTM
  - Using CNN on top of LSTM
- The trained models are also provided for both methods.
  - model_LSTM.h5
  - model_LSTM_CNN.h5
